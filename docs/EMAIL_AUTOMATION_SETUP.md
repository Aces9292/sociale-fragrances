# Email Automation Setup Guide

This guide provides a step-by-step process for setting up automated wholesale email campaigns using Apollo.io.

## Part 1: Tool Setup

### 1. Apollo.io Account Creation
- Visit Apollo.io's [signup page](https://apollo.io/signup) and create a free account.
- Verify your email address and log in to Apollo.io.

### 2. Connecting Gmail/Outlook
- Navigate to 'Settings' in the Apollo.io dashboard.
- Click 'Mail Accounts' and choose either Gmail or Outlook.
- Follow the prompts to authenticate and connect your email account.

### 3. Uploading Prospect List
- Prepare a CSV file of your prospect list with columns (e.g., Name, Email, Company).
- Go to 'Leads' → 'Upload CSV'.
- Map the CSV columns to Apollo.io fields and upload.

### 4. Setting Up Sender Profile
- In 'Settings', select 'Sender Profiles'.
- Create a new sender profile with your name, email, and any personalization.

## Part 2: Sequence Creation

### 1. Creating a 4-Step Email Sequence
- Navigate to 'Sequences' and click 'Create New Sequence'.
- Name your sequence and add stages, specifying content and timing.

### 2. Timing Settings
- Day 1: Introduction email.
- Day 5: Follow-up email.
- Day 10: Reminder email.
- Day 20: Last chance email.

### 3. Personalization Variables
- Use variables like {{first_name}}, {{company}} in your emails.

### 4. Subject Line A/B Testing Setup
- In the sequence editor, use the 'A/B Testing' feature to try different subject lines.

## Part 3: Email Content
- Import the email templates from your document.
- Adapt them using Apollo.io's variables and formatting.

## Part 4: Launch Settings

### 1. Daily Send Limits
- Set limits to 20-30 emails/day to avoid spam flags.

### 2. Reply Handling Rules
- Configure in 'Settings' under 'Reply Handling'.

### 3. Unsubscribe Management
- Use the built-in unsubscribe management tools.

### 4. Tracking Setup
- Enable open, click, and reply tracking in 'Campaign Settings'.

## Part 5: Response Management

### 1. Tagging Interested Prospects
- Tag interested leads in the 'Leads' section for follow-up.

### 2. Booking Meeting Links
- Integrate Calendly for easy meeting bookings.

### 3. Sample Request Workflow
- Set up automated workflows for sample requests.

### 4. CRM Integration
- Optional integration with CRM tools for enhanced management.
