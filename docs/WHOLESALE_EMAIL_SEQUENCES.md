## Automation Rules

### If Reply Contains:
- "interested" / "pricing" / "catalog" → Send line sheet, schedule call
- "samples" / "see them" → Send samples immediately  
- "not interested" / "pass" → Mark as declined, stop sequence
- "too expensive" → Send value comparison, offer opening discount
- "not right now" → Pause 3 months, add to re-engagement list
- "forwarded" → Note contact change, restart sequence with new person

### Follow-up Timing:
- No reply after Email 1 → Email 2 (Day 5)
- No reply after Email 2 → Email 3 (Day 10)
- No reply after Email 3 → Email 4 (Day 20)
- Reply to any email → Exit sequence, manual follow-up
